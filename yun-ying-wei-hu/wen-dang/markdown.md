# Markdown



#### *KeyWord: Markdown字体加粗失效*

- **描述：** 要加粗的文本前后添加了\*\*文本。\*\*，但是不加粗显示
- **原因：** 如果加粗结束的\*\*前是标点，结束后要加一个空格
- **解决办法：** 加粗结束的\*\*后加空格
